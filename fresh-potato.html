Fresh Potatoes

Efficient movie searching database solution

Overview

Developed by a team of six students to provide a simplified and more efficient movie searching process for any users that watches movies and TV shows at any age range. Fresh Potato aims to solve the problem by implementing more defined and detailed recommendation options as well as filtering the movies and shows based on the streaming platforms the users are on with the help of Django web framework and TMDB (The Movie Database) API.

Tech tools

Issues

Though the recommendation option hs been implemented on most of the streaming services, many users still feel lost and overwhelmed while searching for something new to watch due to the limited content and overly complicated designs on most of the modern streaming services and databases.

Study finds more than half of U.S. households are subscribed to multiple streaming services.

Solution

To help with the overwhelming searching process between different streaming services, this project offers one website with more defined and detailed recomendation options for the users before redirecting them to the properly licensed streaming service.

Features

User profile

User profile feature can be used for any registered users. Any changes or updates on users' profile information will be saved and presented everytime they login. User preferences are used for our recommendation/filtering algorithm to accurately display movies tailored to each individual users.

Recommendation algorithms

All users have the option to complete a survey to lay a foundation for the recommendation algorithms. The algorithm takes a complete set of variables into account for a tailored experience on our website, including genres, actors, streaming services, date of birth, and many more.

Movie info

This application utilizes API from TMDb to display the information for the movies. In the current iteration, we are able to retrieve movie information including movie genres, poster, overview, and details including title, rating, budget, popularity, production and more.
